x_64 表示 x(128) 的高 64 位

x(128)' = x + x_64 * 2^64
y(128)' = y + y_64 * 2^64

x' * y' = xy + x * y_64 * 2^64 + y * x_64 * 2^64 + x_64 * y_64 * 2^128
2^128 溢出，因此不用管

	sarq $63, %rcx		; %rcx = x_64

	addq	%rdx, %rcx	; %rcx = y * x_64 + x * y_64
